```md
- ID：Txxxx

- 前置条件：
- 计划修改的文件：
- 明确不做：

### 测试计划

- 单元测试：
- 集成测试：
- 人工烟雾测试：

### 约束门禁

- 需要新建或更新的规范：
- 必须覆盖的规则：
- 是否需要独立约束 PR（docs-only / config-only）：

### 完成结果

- 实现摘要：
- 测试结果：
- 设计偏差：无 / 说明
- 完成 PR：
- 完成日期：
- 下一任务：
```
